1. 관리자 보일러플레이트 프로젝트
Next.js 15 기반의 프로덕션 레디 풀스택 템플릿으로, 스타트업과 개발자를 위한 MVP 개발 가속화 도구입니다.
인증, 대시보드, 리치 텍스트 에디터 등 핵심 기능을 즉시 활용하여 개발 기간을 약 2주로 단축할 수 있습니다.

## 핵심 기술 스택
- Frontend: React 19, Next.js 15, TypeScript 5
- Backend: Supabase (PostgreSQL + RLS 정책), Server Actions
- UI/UX: Tailwind CSS v4, shadcn/ui 커스터마이징
- 빌드 & 최적화: Turbopack, Pretendard Variable 폰트

## 주요 기능
[인증 시스템]
- 이메일/비밀번호 + Google OAuth 로그인
- 비밀번호 재설정, 프로필 관리, httpOnly 세션 쿠키
- Tiptap 리치 텍스트 에디터
- 이미지 업로드 최적화 (Blob URL + Supabase Storage)
- 유튜브 임베딩, 링크, TaskList, 하이라이트 지원
- HTML sanitization으로 XSS 방어

[대시보드 & CRUD]
- 게시물 생성/수정/삭제 + 이미지 자동 정리
- Skeleton 로딩, 에러 바운더리, 고정 게시물 기능

## 개발 포인트
- Server Actions 중심 아키텍처로 API 최소화
- useActionState로 폼 상태 단순화
- Zod 검증으로 서버·클라이언트 이중 유효성 확보
- 반응형 + 다크모드 + 토스트 피드백으로 UX 강화

## 결과
2025년 웹 표준을 반영한 확장성과 보안성이 뛰어난 관리자 보일러플레이트로, 빠른 프로토타이핑과 실서비스 구축을 모두 지원합니다.

2. 달력 예약 앱 프로젝트

개인 레슨 일정 관리용 달력형 웹앱으로, 사용자가 쉽고 직관적으로 예약 일정을 등록·조회·관리할 수 있도록 설계했습니다. 
Supabase + Next.js + Vercel 기반으로 개발 및 배포되어, 모바일과 PC 모두에서 자연스러운 반응형 경험을 제공합니다.

## 핵심 기술 스택
- Frontend: React 19, Next.js 15, TypeScript
- Backend & DB: Supabase (인증 및 일정 데이터 관리)
- UI/UX: Tailwind CSS, DaisyUI
- 배포: Vercel (CI/CD 연동)

## 주요 기능

[회원 관리]
- 회원가입 / 로그인
- 회원정보 수정
- 세션 기반 로그인 유지

[일정 관리]
- 일정 등록, 수정, 삭제
- 달력에서 일정 조회 (PC / 모바일 지원)
- 벌크 일정 등록으로 반복 수업 관리 효율화

## 개발 포인트
- Supabase 인증 + 세션 관리로 안정적인 로그인 유지
- FullCalendar 기반 달력 UI 커스터마이징
- 고객 맞춤형 일정 관리 기능 구현
- 모바일 우선의 반응형 디자인 적용

## 결과
직관적인 UI와 안정적인 데이터 관리로 개인 레슨 일정 운영을 자동화하고, 고객 맞춤형 커스터마이징이 가능한 달력 예약 앱으로 완성되었습니다.
