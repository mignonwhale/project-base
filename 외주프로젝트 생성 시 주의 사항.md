# 외주프로젝트 생성 시 주의 사항

## gitignore 생성

### 1. 전역 gitignore 설정
git config --global core.excludesfile "$HOME\.gitignore_global"

### 2. 파일 생성 및 내용 추가
@"
CLAUDE.md
/dev-docs
*.private.*
.personal/
"@ | Out-File -FilePath $HOME\.gitignore_global -Encoding UTF8

### 3. 확인
Get-Content $HOME\.gitignore_global
