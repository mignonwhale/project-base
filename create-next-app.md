# Next.js 프로젝트 생성

## 개발환경 

- Next.js
- Typescript
- yarn


## 환경세팅

### 프로젝트 세팅
- package.json, node_modules 자동 생성
```
yarn create next-app your-project-name --typescript
```

## yarn 로컬 업그레이드
- `yarn` 버전 올림 
```
  "packageManager": "yarn@4.9.2"
```

- `yarn` 실행: 자동으로 , .yarnrc.yml 생성됨.
```
yarn
```
